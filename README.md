# 学習記録アプリ

## 概要
React + Spring Boot + H2 Database を使用して作成した学習記録管理アプリです。

学習内容、学習時間、学習日を登録・閲覧・更新・削除できます。

## 使用技術

- React
- Spring Boot
- H2 Database
- JPA
- GitHub

## 主な機能

- ログイン
- 学習記録一覧表示
- 学習記録詳細表示
- 学習記録登録
- 学習記録更新
- 学習記録削除
- ログアウト
